# Menentukan bilangan ganjil dan genap
cara kerja pemograman ini yaitu si pengguna disuruh memasukan angka random lalu programnya akan menentukan bahwa angka itu ganjil atau genap.
disini saya mengetik (bilangan 1 % 2 == 0) yaitu dalam artian bahwa bilangan yang bisa dibagi dengan 2 adalah bilangan genap.
